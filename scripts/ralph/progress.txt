## Codebase Patterns
- Auth endpoints (register/login/me) must return all user fields the frontend needs — update the `select` in all three when adding new user fields
- Stripe webhook needs raw body — registered before express.json() in index.ts
- TypeScript strict mode: use `import type { Foo }` for type-only imports
- Lob package has no types — use `// @ts-ignore` before import
- Lob address verification: `lob.usVerifications.verify()` returns deliverability status
- Admin routes use `requireAdmin` middleware from auth.ts (checks `isAdmin` on User model)
- When marking a note as SENT, create a Notification for the sender
- Prisma `{ increment: 1 }` / `{ decrement: 1 }` for atomic counter updates
- Stripe Invoice.charge is not typed — use `(paid as any).charge` with fallback to `paid.id`
