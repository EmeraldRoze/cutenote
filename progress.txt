## Codebase Patterns
- Monorepo: apps/api (Node+Express+TS), apps/web (React+Vite+Tailwind), packages/shared
- All API responses: { data: ... } on success, { error: "plain English message" } on failure
- Auth: JWT in localStorage as 'cn_token', sent as Bearer token, 30-day expiry
- Database: Prisma ORM, all IDs are UUIDs
- Env vars loaded via dotenv in apps/api/src/index.ts
- Frontend proxies /api/* to localhost:4000 via Vite config
- Rate limiting on all /auth/* routes (10 req/min per IP)
